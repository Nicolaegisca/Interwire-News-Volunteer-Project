# Interwire News – Volunteer Project

This repository shows part of my volunteer web development work for **Interwire News**.

The site was based on the **TT5 WordPress theme**, and one of the most difficult parts of the project was fighting the default theme CSS. I had to spend a lot of time overriding built-in styles, fixing layout issues, improving spacing, and making the ticker section work properly on both desktop and mobile.

This project helped me improve my skills in:
- CSS debugging
- theme customisation
- responsive design
- front-end problem solving
- working with WordPress theme limitations

Live website: https://interwire.news/
